Task 3 Method Explanation

1) Start simulated turtle with ros2 run turtlesim turtlesim_node
2) Open a new terminal
3) Control turtle with keys using ros2 run turtlesim turtle_teleop_key
4) Center the turtle and clear the pen with ros2 service call /clear std_srvs/srv/Empty
5) Orient the turle rotating 90 degress using the keys and move forward using the up arrow
6) Close the square with the Turtle
7) Square done
